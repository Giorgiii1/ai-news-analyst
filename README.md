# 🤖 AI News Analyst (RAG)

ეს არის ხელოვნურ ინტელექტზე დაფუძნებული ანალიტიკური ხელსაწყო. აპლიკაცია ეძებს უახლეს ამბებს ნებისმიერ თემაზე და პასუხობს თქვენს კითხვებს რეალურ დროში მოპოვებული ინფორმაციის საფუძველზე.

## 🛠️ ფუნქციები
- **Real-time News:** იყენებს NewsAPI-ს უახლესი ამბების მოსაძიებლად.
- **RAG Architecture:** იყენებს ChromaDB-ს ინფორმაციის დასამუშავებლად.
- **Smart AI:** პასუხებს აგენერირებს Google Gemini AI.
- **Hybrid Knowledge:** აერთიანებს სტატიების ანალიზს და AI-ს ზოგად ცოდნას.
